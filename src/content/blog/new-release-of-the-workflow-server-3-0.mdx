We are really excited to announce that we just made a new release of our **Workflow Server 3.0**.

![Workflow Server 3.0 dark theme](/images/blog/new-release-of-the-workflow-server-3-0/workflow-server-30-dark.jpg "Workflow Server 3.0 dark theme")

### And here is a list of changes we made:

-   Redesigned and updated the style of the entire interface — it now looks super fresh. And we also added a **dark theme!**
-   Now you have only **the most important** things on the Dashboard — running processes, schemes, plug-ins, forms.
-   Workflow API, Callback API, Schemes and Forms pages have also been reorganized — check them out, they look **completely** different now!
-   The process of creating forms has become more **convenient and understandable**, you should try!
-   Added **Settings section** — it contains rarely edited server settings that used to be on the Dashboard.
-   The saving process has become more **intuitive and easier**. Changes can now be canceled (Reset) before being sent to the server.

... and many more!
